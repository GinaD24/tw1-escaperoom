LA MANSIÓN MISTERIOSA

Etapa: Lobby

Resuelve el siguiente acertijo para pasar a la próxima etapa

La puerta hacia la siguiente habitación está bloqueada por un candado,
busca la clave en este acertijo.

En este hall hablan retratos y susurran las paredes,
pero hay algo que todo importa: sin ruido no se revelan claves.

Soy lo que reina cuando nadie dice nada;
quien me nombra, se rompe.

¿Qué soy?
